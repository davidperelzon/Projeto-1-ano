# Projeto-1-ano
🚧 No More Holes

Você já passou por um buraco na rua e pensou: “Alguém precisa resolver isso”?

O No More Holes nasceu exatamente para transformar essa indignação em ação. Nossa plataforma permite que qualquer cidadão denuncie buracos nas vias públicas de forma rápida, simples e eficiente — ajudando a tornar a cidade mais segura para motoristas, ciclistas e pedestres.

Com poucos cliques, você pode:
✔ Registrar a localização do problema
✔ Enviar fotos
✔ Acompanhar o status da solicitação
✔ Contribuir para uma cidade mais organizada

Além disso, o projeto propõe uma solução inovadora: o uso de tecnologia com drones para aplicação emergencial de material em buracos, reduzindo riscos enquanto o reparo definitivo não é realizado.

O No More Holes não é apenas um aplicativo — é um movimento de cidadania ativa, inovação urbana e colaboração comunitária.
